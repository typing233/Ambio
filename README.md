# Ambio – 沉浸环境音

一款多轨环境音网页应用，无需外部音频文件，所有声音通过 Web Audio API 实时合成。

## 功能

- **4 个场景**：森林 🌲 · 咖啡厅 ☕ · 雨天 🌧️ · 深夜书房 🕯️
- **智能匹配**：根据当前时间段、天气、心情自动推荐最合适的场景
- **多轨混音**：每个场景包含 1–2 条持续底层音 + 多条随机触发细节音效，可独立调节音量 / 静音
- **无缝循环**：所有音效基于振荡器与噪声发生器合成，无采样断点
- **定时关闭**：15 / 30 / 45 / 60 / 120 分钟定时自动停止
- **播放 / 暂停**：全局控制，切换场景自动衔接

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 技术栈

- **Vite + React + TypeScript**
- **Tailwind CSS v4**（`@tailwindcss/vite` 插件）
- **Web Audio API**（程序化音频合成，无外部音频依赖）
